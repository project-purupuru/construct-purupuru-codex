# The HENLO System

The foundational architecture unifying Project Purupuru. Each HENLO letter corresponds to a personality pairing, a Wuxing element, a Confucian ethical principle, a bear classification, and a chromatic identifier.

## What HENLO Connects

HENLO structures everything:
- Personality pairings (caretaker trait + puruhani trait)
- Elemental assignments (Wuxing)
- Ethical concepts (Confucian virtues)
- Chromatic language (color palette)
- Bear classifications (bera stickers)
- Jani elemental variants (Wuxing core)

It is a greeting, a lifestyle, an ethos, and for some, a belief system.
